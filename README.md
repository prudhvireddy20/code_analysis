Clone the repository to your local machine: https://github.com/prudhvireddy20/code_analysis

To start the frontend application: Navigate to frontend directory
    --cd frontend

Install the necessary Node.js dependencies:
    --npm install

Start the development server:
    --npm run dev

Running the Backend
open new terminal
ensure in the code_analysis directory and navigate to backend directory
    --cd backend

Create and activate a Python virtual environment:
python3 -m venv venv
source venv/bin/activate

Install the requirements by using the 
    pip install -r requirements.txt

Run the script by using the 
    python3 app.py
